SimpleApiServer
📖 Описание проекта

SimpleApiServer — это минималистичный RESTful API сервер, написанный на чистом C# без внешних зависимостей. Реализует полный CRUD для управления задачами ToDo-листа с использованием встроенного HttpListener.
🎯 Основная идея
Создать самодостаточный сервер задач, который:

    Запускается одной командой dotnet run

    Работает на http://localhost:5000

    Поддерживает все REST операции (GET/POST/PUT/DELETE)

    Хранит данные в памяти (2 предустановленные задачи)

    Имеет понятные JSON ответы и HTTP статусы

🚀 Ключевые возможности
📋 /api/tasks        → GET все задачи
🔍 /api/tasks/1      → GET задача по ID  
➕ /api/tasks        → POST новая задача
✏️  /api/tasks/1      → PUT обновить задачу  
🗑️  /api/tasks/1      → DELETE удалить задачу

💡 Зачем это нужно?

    Обучение: пример реального REST API на C#

    Прототипирование: быстрый бэкенд без фреймворков

    Тестирование: база для Postman/cURL/HTTP клиентов

    Демо: показывает возможности HttpListener
